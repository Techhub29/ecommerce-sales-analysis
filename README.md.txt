# 📊 E-Commerce Sales Analysis

## 📌 Overview
This project analyzes e-commerce transaction data to identify sales trends, top-performing products, and customer behavior.

---

## 🔧 Tools Used
- Python (Pandas)
- Tableau

---

## 📊 Key Insights
- Sales peak in November, indicating strong seasonal demand
- Decorative products contribute the majority of revenue
- Revenue is concentrated among a small number of high-value customers
